مساعد ذكي

تحدّث مع طبيبك الذكي — بالصوت أو الكتابة

مساعد صحي مدعوم بالذكاء الاصطناعي يفهم تاريخك الطبي بالكامل: أدويتك، تحاليلك، أعراضك، علاماتك الحيوية، وتغذيتك. اسأله أي سؤال صحي واحصل على إجابات مخصصة لحالتك.

  • محادثات صوتية كاملة — تكلم ويجيبك بالصوت بالعربي والإنجليزي
  • فحص أعراض ذكي — أدخل أعراضك واحصل على تقييم فوري مع مستوى الخطورة
  • كشف تفاعلات الأدوية — يحذّرك تلقائياً من تعارض أدويتك
  • رؤى صحية يومية وتحليل اتجاهات ٩٠ يوماً مدعوم بالذكاء الاصطناعي
تتبع صحي شامل

صوّر تحاليلك واترك الباقي علينا

ماسح تحاليل بتقنية OCR الذكية — صوّر ورقة التحليل وسيستخرج التطبيق كل النتائج تلقائياً مع تحليل ذكي يوضح لك القيم الطبيعية وغير الطبيعية. بالإضافة لمتابعة أدويتك وعلاماتك الحيوية وكل شيء آخر.

  • ماسح تحاليل OCR — صوّر وتحصل على النتائج + تحليل ذكي فوري
  • متابعة أدوية مع تنبيهات المواعيد، عدّاد المخزون، وسجل الجرعات
  • علامات حيوية: ضغط، نبض، سكر، حرارة، وزن، أكسجين — مع رسوم بيانية
  • سجل تطعيمات مع تنبيهات الجرعات القادمة والمتأخرة
عافية وأسلوب حياة

تغذية، تمارين، نوم، ودورة — كل شيء متصل

تابع كل جانب من جوانب صحتك اليومية. سجّل وجباتك مع السعرات والبروتين، تمارينك مع حساب السعرات المحروقة تلقائياً، نومك مع جودة النوم، ودورتك الشهرية مع تنبؤات ذكية — كل البيانات تغذي الذكاء الاصطناعي لنصائح أدق.

  • تتبع تغذية: بحث أطعمة، سعرات، بروتين، كربوهيدرات، دهون — مع ملخص أسبوعي
  • تمارين رياضية: ١٥ نوع تمرين، حساب سعرات بنظام MET، تحليلات أسبوعية
  • تتبع نوم: وقت النوم والاستيقاظ، جودة النوم، توزيع أسبوعي
  • متابعة دورة شهرية: تسجيل الدورة، أعراض، تنبؤات الدورة القادمة وفترة الخصوبة
  • مزامنة مع Apple Health وأجهزة اللياقة القابلة للارتداء

صحة عائلتك تحت عينك — وتواصل مباشر مع طبيبك

أنشئ دائرة عائلية وتابع صحة أحبائك: أدويتهم، تحاليلهم، علاماتهم الحيوية. تواصل مع أفراد عائلتك بالرسائل الفورية. بالإضافة لذلك، يمكن لطبيبك الوصول لبياناتك بإذنك، إضافة أدوية وتحاليل، والتواصل معك مباشرة.

  • دائرة عائلية: ادعُ أفراد عائلتك بكود أو رابط، مع مستويات صلاحية مرنة
  • رسائل فورية مع العائلة والأطباء داخل التطبيق
  • شبكة مقدمي خدمات: مختبرات، عيادات، صيدليات — مع تقييمات ومسافات
  • طلب إعادة صرف الأدوية من الصيدليات القريبة

جواز صحي وأمان

جواز صحي PDF احترافي — وحماية بأعلى المعايير

صدّر تقريراً طبياً شاملاً بصيغة PDF يتضمن كل بياناتك الصحية: معلوماتك، أمراضك، حساسياتك، أدويتك، تحاليلك، تطعيماتك، وعلاماتك الحيوية — مع رمز QR للطوارئ. كل هذا محمي بتشفير كامل ومصادقة بيومترية.

  • تقرير PDF ثنائي اللغة — عربي وإنجليزي بتصميم طبي احترافي
  • رمز QR للطوارئ — يحتوي معلوماتك الصحية الأساسية للمسعفين
  • مصادقة بيومترية (Face ID / Touch ID) وتشفير كامل للبيانات
  • بياناتك ملكك — لا نبيعها ولا نشاركها أبداً. متوافق مع GDPR

حمّل وسجّل

حمّل التطبيق مجاناً من App Store. سجّل بإيميلك أو عبر Apple/Google — خلال ثوانٍ تكون جاهز.

2

أضف بياناتك الصحية

صوّر تحاليلك، أضف أدويتك، سجّل علاماتك الحيوية، فعّل Apple Health — التطبيق يبني ملفك الصحي الشامل تلقائياً.

3

اسأل وتابع واطمئن

اسأل المساعد الذكي أي سؤال صحي، تابع اتجاهاتك الصحية، احصل على رؤى يومية، وشارك جوازك الصحي مع طبيبك.
