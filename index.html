تحدّث مع طبيبك الذكي — بالصوت أو الكتابة

مساعد صحي مدعوم بتقنية Claude AI يفهم تاريخك الطبي بالكامل: أدويتك، تحاليلك، أعراضك، علاماتك الحيوية، وتغذيتك. اسأله أي سؤال صحي واحصل على إجابات مخصصة لحالتك — بالعربي أو الإنجليزي.

  • محادثات صوتية كاملة — تكلم ويجيبك بالصوت فوراً
  • فحص أعراض ذكي مع تقييم مستوى الخطورة والإجراءات المقترحة
  • كشف تفاعلات أدوية تلقائي — يحذّرك قبل الخطر
  • رؤى صحية يومية + تحليل اتجاهات ٩٠ يوماً بالذكاء الاصطناعي
تتبع صحي شامل

صوّر تحاليلك — والذكاء الاصطناعي يتولى الباقي

ماسح تحاليل بتقنية OCR يستخرج كل النتائج من صورة الورقة تلقائياً مع تحليل ذكي يميّز القيم الطبيعية وغير الطبيعية. أضف لذلك: متابعة أدوية مع تنبيهات وعدّاد مخزون، علامات حيوية مع رسوم بيانية، وسجل تطعيمات كامل.

  • ماسح تحاليل OCR — صوّر ورقة التحليل واحصل على النتائج + تحليل ذكي
  • متابعة أدوية مع تنبيهات المواعيد وعدّاد المخزون وسجل الجرعات
  • ٦ أنواع علامات حيوية مع رسوم بيانية واتجاهات تاريخية
  • سجل تطعيمات مع تنبيهات الجرعات القادمة والمتأخرة
عافية وأسلوب حياة

تغذية، تمارين، نوم، ودورة شهرية — كل شيء متصل

سجّل وجباتك مع البحث في قاعدة الأطعمة والسعرات والماكروز. تابع ١٥ نوع تمرين مع حساب السعرات المحروقة تلقائياً. راقب نومك مع جودة النوم. تابع دورتك الشهرية مع تنبؤات ذكية للدورة القادمة وفترة الخصوبة — كل البيانات تغذي الذكاء الاصطناعي لنصائح أدق.

  • تغذية: بحث أطعمة، سعرات، بروتين، كربوهيدرات، دهون + ملخص أسبوعي
  • تمارين: ١٥ نوع، حساب سعرات بنظام MET، تحليلات أسبوعية حسب النوع
  • نوم: وقت النوم/الاستيقاظ، تقييم الجودة، توزيع أسبوعي
  • دورة شهرية: تسجيل الدورة والأعراض، تنبؤ الدورة القادمة وفترة الخصوبة
  • مزامنة مع Apple Health + لوحة بيانات الأجهزة القابلة للارتداء

صحة عائلتك تحت عينك — وتواصل مباشر مع طبيبك

أنشئ دائرة عائلية وتابع صحة أحبائك بالكامل. تراسل مع العائلة والأطباء فوراً. شبكة مقدمي خدمات تشمل مختبرات وعيادات وصيدليات. اطلب إعادة صرف أدويتك من أقرب صيدلية. واحجز مواعيد المختبر بضغطة زر.

  • دائرة عائلية مع دعوات بكود/رابط ومستويات صلاحية (عرض، إدارة، كامل)
  • رسائل فورية مع العائلة والأطباء + إشعارات دفع
  • دليل مختبرات وعيادات وصيدليات مع تقييمات وترتيب حسب المسافة
  • طوارئ SOS: مشاركة بياناتك الصحية مع جهات الاتصال بضغطة واحدة

جواز صحي PDF احترافي — وبياناتك محمية بالكامل

صدّر تقريراً طبياً شاملاً بصيغة PDF يتضمن: معلوماتك، أمراضك، حساسياتك، أدويتك، تحاليلك (مع تمييز القيم غير الطبيعية)، تطعيماتك، علاماتك الحيوية، وملخصات العافية — مع رمز QR للطوارئ. التقرير ثنائي اللغة ويمكن مشاركته عبر AirDrop أو الإيميل أو الطباعة.

  • نقاط صحة من ١٠٠ محسوبة من ٥ عوامل: علامات حيوية، أدوية، تحاليل، نشاط، عافية

  • Face ID / Touch ID + تشفير كامل + تخزين آمن للتوكنات

حمّل وسجّل

3

اسأل واطمئن

اسأل المساعد الذكي أي سؤال، تابع اتجاهاتك، احصل على رؤى يومية، وشارك جوازك الصحي مع طبيبك.

أسئلة شائعة

أسئلة شائعة

نعم، صحتك مجاني بالكامل للتحميل والاستخدام. لا توجد إعلانات ولا نبيع بياناتك أبداً. يتوفر اشتراكات اختيارية للمزايا المتقدمة مثل محادثات ذكاء اصطناعي غير محدودة والوضع الصوتي.

المساعد مدعوم بتقنية Claude AI ويفهم سياقك الطبي الكامل: أدويتك، تحاليلك، أمراضك، علاماتك الحيوية، وبيانات العافية. يمكنك التحدث معه بالصوت أو الكتابة بالعربي والإنجليزي. يقدم فحص أعراض، كشف تفاعلات أدوية، تحليل اتجاهات ٩٠ يوماً، ورؤى صحية يومية.

نعم! صوّر ورقة التحليل وسيستخرج التطبيق كل النتائج تلقائياً بتقنية OCR. ثم يحلل الذكاء الاصطناعي كل قيمة ويميّز القيم الطبيعية وغير الطبيعية ويقدم تفسيراً طبياً تفهمه.

بالتأكيد. نستخدم تشفيراً كاملاً ومصادقة بيومترية (Face ID / Touch ID) وتخزيناً آمناً للتوكنات. بياناتك لا تُباع ولا تُشارك مع أي طرف ثالث أبداً. التطبيق متوافق مع GDPR ويدعم تصدير البيانات وحذف الحساب بالكامل.

نعم. التطبيق مبني بمعمارية offline-first — بياناتك الصحية محفوظة محلياً فيمكنك عرض أدويتك وعلاماتك الحيوية وتحاليلك بدون إنترنت. الطلبات الفاشلة تُحفظ تلقائياً وتُرسل عند عودة الاتصال.
